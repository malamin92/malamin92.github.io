The CSS Box Model

Padding, Border and Margin

11/04/2015

The CSS box model is a very interesting feature of CSS. Essenetially what the Box Model is refers to the invisible box that surounds every HTML element there is. It referes to the padding, border and margin that surrounds the main 'content' that resides in an element. This is a really neat part of HTML and CSS because what it does is allow a designer to maniputlate the spacing between elements to the tee. Each of the three properties of the box model allows for adjustment of the spacing of the content within the HTML body. It determines how the content will ultimately sit on that page in relation to the other elements that surround it. I think the best way to get a sense of the box model is to see the visual:

As you can see from the image above, the 'content' of an element sits in the middle of the element. Then going from the inside out, there is the padding -> border -> margin. When trying to remember which spacing each refers to, I try to think of them in that order.

Padding refers to the space that comes right after the content, between the border and the content. In order to demonstrate, I am going to create a div with an embeded p element "SAMPLE CONTENT", and create space between the content and green border.

At this point, the border property should be self-explanitory. It is simply the border that surrounds the content in repsect to the margin. The borders width can be adjusted to make it as thick as we want; the style and color can also be changed. Here is the first example but with a thicker border:

The last part of the Box-Model I wil talk about is the margin. The margin is the space between the borders of two elements. When thinking about elements that have no borders, I just imagine there is an invisible border because that is what the Box-Model implies, that every element has a box around it. That is basic difference between the properties of the Box Model. You can alter the padding, boder and margin of elements in order to space out elements on your page in any way desireable. I hope you can use this knowledge next time you are trying to space the elemets on your page. This can be a confusing idea at first but you will get the hang of if you just play around with the properties a bit.
