SQL Cheat Sheet

SQL (Structured Query Language) is a special-purpose programming language designed for managing data held in a relational database management system (RDBMS), or for stream processing in a relational data stream management system (RDSMS).

OPENING A SQLITE3 DATABASE

Open sqlite3 database in console:

sqlite3 database_name.sqlite3

DATABASE CREATION

Create a new database:

CREATE TABLE table_name (col1 type, col2 type, etc...);

Insert data into tables:

INSERT INTO table_name (col1, col2) VALUES (val1, val2);

Update columns:

UPDATE table_name SET col_name = val WHERE condition;

Alter tables, add or drop columns:

ALTER TABLE table_name ADD col_name type;
ALTER TABLE table_name DROP COLUMN col_name;

BASIC QUERIES

Get all columns from a table:

SELECT * FROM table_name;

Get columns depending on a condition:

SELECT * FROM table_name WHERE condition;

Get columns within a range:

SELECT col_name FROM table_name WHERE condition
BETWEEN val1 AND/OR val2 ;

Get columns in a ascending/descending order:

SELECT col_name FROM table_name
ORDER BY col DESC/ASC;

Limit results of query:

SELECT col_name FROM table_name
WHERE condition LIMIT number_to_limit_to;

AGGREGATE QUERIES

Get the number of all rows

SELECT COUNT(*) FROM table_name WHERE condition;
